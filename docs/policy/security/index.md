# Overview

This site is the core end-user documentation for our public cloud providers at FWH. You will find dedicated sections about all main providers we are working with, where we share guidelines around:

- How to start using public cloud platforms ?
- Security best practices
- Network connectivity with FWH ecoystem
- Cost optimizations

## Officially Supported Providers

Rome currently supports three hosting providers:

- Amazon Web Services
- Microsoft Azure
- I3D.net

Support in this context would mean that Rome teams are working to make their services available on these platforms and you can get architecture or technical supports from IT teams. But also that there are established processes for procurement, we have regular contacts with account managers/solutions architects and have an Enterprise Agreement (EA) established. You can check our dedicated section about Enterprise Agreements for [AWS](./getting-started/public-cloud/contracts/aws-contract.md), [Azure](./getting-started/public-cloud/contracts/azure-contract.md) and [I3D.net](./getting-started/i3d.net/index.md)

!!! info "Attention"
    Please note that Google cloud support is being deprecated, with an end of contract in 2024
